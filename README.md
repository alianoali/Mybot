# بوت الذكاء الاصطناعي - تسجيل دخول Google

واجهة تسجيل دخول احترافية تسمح للمستخدمين بالدخول عبر حساب Google فقط، مع إنشاء حساب تلقائي في النظام.

## ✨ الميزات
- تصميم عربي عصري (RTL)
- واجهة محمية وأمان عالي
- يدعم فقط حسابات Gmail (@gmail.com)
- رسومات متحركة وجذابة
- جاهز للتكامل مع Backend

## 🔧 المتطلبات
- Client ID من Google Cloud Console
- تشغيل عبر خادم محلي (Live Server)

## 🚀 كيفية التشغيل
1. شغّل الملف عبر Live Server (VS Code) أو أي خادم محلي
2. تأكد من إضافة `http://localhost` إلى Authorized JavaScript origins في Google Cloud Console
